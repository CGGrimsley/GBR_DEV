## ü§ù How to Contribute

### 1. Fork & Clone
```bash
git clone https://github.com/CGGrimsley/GBR_DEV.git
cd GBR_DEV
```

### 2. Create a Branch
```bash
git checkout -b feature/your-feature-name
```

### 3. Make Your Changes
- Follow existing code style
- Test thoroughly in-game
- Comment your code clearly
- Update documentation if needed

### 4. Commit Your Changes
```bash
git commit -m "feat: Add your feature description"
```

### 5. Submit a Pull Request

**For Code Contributions:**
```markdown
## Description
[Describe your changes]

## Testing
[How did you test this? ]

## CLA Agreement
I agree to assign copyright of this contribution to kdrgold under the terms of the LICENSE file.
```

**For Asset Contributions (Option B - Retained Ownership):**
```markdown
## Description
[Describe your asset]

## Asset License Choice
I choose **Option B:  Retained Ownership**

I retain copyright of the following assets:
- `path/to/my_texture.png` - Created by [Your Name]
- `path/to/my_model.json` - Created by [Your Name]

I grant Gold's Beasts Rebalance a perpetual, non-exclusive, non-revocable license to use these assets as described in LICENSE Section 4.1 Option B. 

## Asset Credits
Please credit as:  [Your Name or Username]

## Testing
[How to verify the asset works in-game]
```

## üéØ Contribution Guidelines

### What We're Looking For
- üêõ Bug fixes
- ‚öñÔ∏è Balance improvements
- ‚ú® New elite variants
- üé® Asset improvements (textures, models, icons)
- üìù Documentation updates
- üß™ Testing and feedback
- üí° New mechanics and features

### Asset Contribution Tips
- **Models**: Follow existing GBR model structure
- **Textures**:  Match the Hytale art style
- **Icons**: 64x64 or 128x128 PNG with transparency
- **Sounds**: OGG format, appropriate volume levels
- **Configs**: JSON format, well-commented

### What We're NOT Looking For
- Major gameplay overhauls without prior discussion
- Features that conflict with project goals
- Contributions that violate Hytale's EULA
- Low-quality or untested changes
- Assets ripped from other games/mods

## üîç Review Process

1. **Submission**: You create a Pull Request

3. **Code/Asset Review**:  Maintainers review your contribution
4. **Discussion**: We may request changes or clarifications
5. **Testing**: Changes are tested in-game
6. **Merge**: If approved, your contribution is merged
7. **Credit**: You're added to the credits list with appropriate attribution

## üõ†Ô∏è Creating Companion Mods

Want to create a separate mod that works with GBR? You can! 

### Companion Mod Guidelines

‚úÖ **You CAN:**
- Create mods that add new elite variants
- Build integration with other creature mods
- Develop config tools or GUIs for GBR
- Make texture/resource packs
- Create extended loot tables
- Build compatibility patches

‚ùå **You CANNOT:**
- Redistribute any files from GBR
- Modify GBR's core files
- Claim affiliation or endorsement
- Include GBR in your mod download

### Requirements: 
1. **Attribution**:  Clearly state "Requires Gold's Beasts Rebalance by kdrgold"
2. **Independence**: Your mod must be a separate download
3. **Compatibility**: List which GBR version(s) you support
4. **Documentation**:  Explain how your mod extends/enhances GBR
5. **License**: Your mod can use any license you choose

### Example Companion Mods:
- **GBR:  Desert Variants** - Adds elite creatures to desert biomes
- **GBR Config Manager** - GUI for adjusting spawn rates and difficulty
- **Enhanced Drops for GBR** - Additional loot table options
- **GBR x [Other Mod]** - Integration between mods

**Need Help?** Contact @kdrgold on Discord to discuss your companion mod idea!
